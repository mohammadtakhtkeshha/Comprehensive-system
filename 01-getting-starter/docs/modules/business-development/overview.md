# توسعه کسب‌وکار (Business Development)

## معرفی

ماژول توسعه کسب‌وکار، مدیریت پروژه‌های بیرونی و ارائه سرویس‌های PSP به سازمان‌ها را بر عهده دارد.

## ویژگی‌های کلیدی

- مشارکت با نهادهای بیرونی
- ارائه سرویس‌های PSP به سازمان‌ها
- سرویس تولید شناسه پرداخت
- سرویس‌های API برای سازمان‌ها
- تخصیص ترمینال به پروژه‌های خاص
- مدیریت SLA بین‌سازمانی
