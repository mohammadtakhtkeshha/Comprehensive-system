# مدیریت پذیرندگان (Merchant Management)

## معرفی

ماژول مدیریت پذیرندگان، هسته اصلی سیستم PSP محسوب می‌شود و تمامی فرآیندهای مربوط به ثبت، احراز هویت و مدیریت پذیرندگان را پوشش می‌دهد.

## ویژگی‌های کلیدی

### 1. ثبت و احراز هویت پذیرنده
- ثبت اطلاعات اولیه پذیرنده
- تکمیل مدارک و مستندات
- احراز هویت از طریق سرویس‌های استعلام
- تایید نهایی و فعال‌سازی

### 2. تغییر اطلاعات پذیرنده
- به‌روزرسانی اطلاعات پایه
- تغییر وضعیت فعالیت
- مدیریت قراردادها

### 3. مدیریت قرارداد و وضعیت فعالیت
- ثبت قراردادهای جدید
- تمدید قراردادها
- تعلیق و لغو قرارداد
- مدیریت وضعیت‌های مختلف پذیرنده

### 4. نگاشت پذیرنده به MID/TID
- تخصیص Merchant ID (MID)
- نگاشت ترمینال‌ها به Terminal ID (TID)
- مدیریت روابط MID-TID

### 5. اتصال به سرویس‌های استعلام
- استعلام از سامانه ثبت احوال
- استعلام از سامانه مالیات
- استعلام از سامانه شاهکار
- استعلام از سامانه‌های بانکی

### 6. مدیریت ضامن‌ها و مدارک
- ثبت اطلاعات ضامن‌ها
- مدیریت مدارک شناسایی
- آپلود و مدیریت فایل‌های مدارک
- اعتبارسنجی مدارک

## ساختار اطلاعاتی

### اطلاعات پذیرنده
- اطلاعات هویتی (نام، کد ملی، شماره ثبت)
- اطلاعات تماس (تلفن، ایمیل، آدرس)
- اطلاعات حساب بانکی
- اطلاعات فعالیت و نوع کسب‌وکار

### اطلاعات قرارداد
- نوع قرارداد
- تاریخ شروع و پایان
- کارمزدها و نرخ‌ها
- شرایط و ضوابط

### مدارک و مستندات
- مدارک ثبت شرکت
- مدارک احراز هویت
- مدارک مالی
- سایر مستندات

## فرآیندهای اصلی

1. **ثبت پذیرنده جدید**
   - دریافت اطلاعات اولیه
   - تکمیل مدارک
   - ارسال برای بررسی

2. **احراز هویت**
   - بررسی مدارک
   - استعلام از سرویس‌های بیرونی
   - تایید یا رد درخواست

3. **فعال‌سازی**
   - تخصیص MID
   - ایجاد قرارداد
   - فعال‌سازی دسترسی‌ها

4. **مدیریت دوره‌ای**
   - به‌روزرسانی اطلاعات
   - تمدید قراردادها
   - بررسی وضعیت
